DARTCRAWL WEB

DART 보고서 링크를
엑셀 파일로 정리합니다.

DART 사업보고서 URL을 붙여 넣으면 서버리스 API가 상세표를 읽고 Excel 파일로 내려줍니다. 화면은 GitHub Pages에, 수집은 Worker에 두는 구조를 전제로 만들었습니다.

입력 `main.do?rcpNo=...` 형식의 DART 링크
출력 여러 시트를 가진 Excel XML `.xls` 파일
대상 표 연결대상 종속회사, 계열회사, 타법인출자 현황

How It Works

동작 방식

  1. 프런트엔드가 Worker에 DART 링크를 전달합니다.
  2. Worker가 DART 본문과 상세표 HTML을 가져옵니다.
  3. 필요한 표를 정리해 Excel 파일로 응답합니다.
왜 Worker가 필요하나

GitHub Pages는 정적 사이트라 브라우저에서 DART를 직접 읽을 때 CORS 제한을 받습니다. 그래서 DART 요청은 별도 서버리스에서 처리해야 합니다.

지원 링크

현재는 `https://dart.fss.or.kr/dsaf001/main.do?rcpNo=...` 형식의 사업보고서 링크를 기준으로 만들었습니다.
